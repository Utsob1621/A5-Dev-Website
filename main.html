Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer = innerHTML ও innerText এবং textContent আর মধ্যে যা পার্থক্য :
1. innerHTML ব্যবহার করে html থেকে একটি element এর html কোডসহ পাওয়া যায়, এবং নতুন element html কোডসহ তৈরি করা যায়।
2. innerText ব্যবহার করে html থেকে একটি element থেকে ওই element এর ভেতরের টেক্সট নেওয়া বা পাওয়া যায়, এবং element এর ভেতরে নতুন টেক্সট অ্যাড করা যায়।
3. textContent ব্যবহার করে html থেকে একটি element এর ভেতরের সব টেক্সট পাওয়া যায়, এমনকি hidden টেক্সটও। এছাড়া element এর ভেতরে নতুন টেক্সট অ্যাড করতেও এটি ব্যবহার করা যায়।

Question-3: What is event delegation in the DOM?

Answer = Event delegation হছে :
1.Event Delegation ব্যবহার করে আমরা html এ কোনো parent element-এ event handler ব্যবহার করে child element এর ইভেন্ট হ্যান্ডল করতে পারি। এইচটিএমএল এ কোনো parent element -এর
যখন অনেকগুলো child element থাকে, এবং প্রত্যেকটির জন্য আলাদা event handler দেওয়া সম্ভব হয় না, তখন event delegation ব্যবহার করা হয়। আমরা event.target ব্যবহার করে বুঝতে পারি কোন
child element -এ event ঘটেছে।

Question-4: What is event bubbling in the DOM?

Answer = Event bubbling হছে : 1.ইভেন্ট বাবলিং হল DOM (Document Object Model) এর একটি মেকানিজম, যেখানে একটি এলিমেন্টে ইভেন্ট ট্রিগার হলে সেই ইভেন্ট DOM ট্রি এর মাধ্যমে উপরের দিকে
বাবল করে (চাইল্ড থেকে প্যারেন্ট এলিমেন্টে)। অর্থাৎ, যদি একটি চাইল্ড এলিমেন্টে ক্লিক করা হয়, তাহলে সেই ইভেন্টটি তার প্যারেন্ট এলিমেন্টেও ট্রিগার হবে।

Question-5: How do you create, add, and remove elements using JavaScript?

Answer = JavaScript? ব্যবহার করে যে ভাবে উপাদান create, add and remove করা যায় :
1. নতুন element তৈরি করা
const paragraph = document.createElement("p");
paragraph.textContent = "এটি একটি নতুন প্যারাগ্রাফ তৈরি করা হলো";

2. নতুন element যোগ করা
const container = document.getElementById("id");
container.appendChild(paragraph);

3. নতুন element বাদ দেওয়া
const oldParagraph = document.getElementById("id");
oldParagraph.parentNode.removeChild(oldParagraph);
